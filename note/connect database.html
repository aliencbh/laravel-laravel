1. ensure you have a database set up and have the credentials (username, password, hostname, etc.) ready
  2. open the config/database.php file and find the connections array
  3. change database name in ['default' => env('DB_CONNECTION', 'mysql'),]
  4. open the .env file
  5. edit these variable
    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=laravel
    DB_USERNAME=root
    DB_PASSWORD=
  6. save the .env file and exit
  7. $users = DB::table('users')->get();
